# Цифровая визитка Ольги

Персональная страница: проекты, навыки, Мей и контакты.

## Файлы

- `index.html` — структура и тексты
- `styles.css` — оформление
- `script.js` — появление блоков, меню, видео, Telegram
- `songs-player.js` / `songs-reactions.js` — плеер и реакции

## Запуск

Откройте `index.html` в браузере.

## Деплой

После правок: commit → push в GitHub → автодеплой Timeweb (HTML/CSS/JS).
