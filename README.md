# Цифровой мир с Мей

Готовая персональная страница-знакомство для Cursor.

## Файлы

- `index.html` — структура страницы и тексты.
- `styles.css` — оформление, адаптивность, анимации.
- `script.js` — мягкое появление блоков при прокрутке.
- `assets/` — изображения Мей и аватарка.

## Как запустить

Открой `index.html` в браузере или перетащи папку проекта в Cursor.

## Где менять

- Цвета: блок `:root` в `styles.css`.
- Тексты: соответствующие секции в `index.html`.
- Карточки проектов: блок `<section id="projects">`.
